example_of_how_to_refer_to_a_file_in_the_package()[source]¶

Method showing how to reference files within the package

You need to refer to the file in the package bundle, because you don’t know where the module will be called from… this gets you the right path always, relative to sourcecodes

Don’t forget to include non-python files in the MANIFEST.in too!
