Documentation As Is¶

This a python package template based on shoulder of

  1. python-library-template as main pakage template created by Tom Clark

  2. Furo (Sphinx theme) for Documentation

  3. Github Pages for hosting Documentation

  4. Pre-commit hooks for cleaning and building things before we ake any commit (Includes Sphinx Auto Doc Builds too)

Lot have to be written and would be written eventually.

Something which I dont what to see right now but is only for reference A Chapter.

Steps post cloning¶

Use following steps to make use the whole template

  1. Do a pre-commit install with pre-commit install

  2. Add your code in respective places

  3. Write documentation

  4. Run sphinx-build -b html -j 2 docsrc docs to create the documentation and sphinx gallery

    1. (Optional) Check for any errors with precommit pre-commit run --all-files

  5. Add your files and do your commit with some good message and wait for pre-commeit to do its charm.

  6. Correct any error if any from precommit.

  7. Push your code to git

  8. Host your documentation on github pages by changing folder to docs

  9. Showcase you work to world. :)
